# Threat Intelligence Analysis

This repository contains my structured analysis of cyber attack scenarios using real-world methodologies.

## What I focus on:
- Mapping attacker behavior to MITRE ATT&CK
- Identifying Indicators of Compromise (IOCs)
- Breaking down attack chains step-by-step
- Evaluating realism in reported threats

## Example Analysis

### Scenario:
An attacker gains access via phishing and deploys malware.

### Analysis:
- Initial Access: Phishing (T1566)
- Execution: Malicious file execution
- Persistence: Registry modification
- Detection Opportunities:
  - Suspicious email patterns
  - Endpoint behavior anomalies
